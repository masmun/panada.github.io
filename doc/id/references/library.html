Library

Library adalah sebuah class yang memiliki tugas spesifik yang fungsinya bisa digunakan oleh controller, model ataupun library lainnya.

Panada telah menyertakan beberapa library yang umumnya paling sering digunakan. Library natif yang disediakan oleh Panada disebut resource. Library natif ini terletak pada folder:

panada/Resources/

Untuk mengetahui fungsi masing-masing library ini dan cara penggunaanya, silahkan lihat di bagian Resources.

Membuat Library

Untuk membantu pembuatan aplikasi, Anda juga bisa membuat library sendiri. Library tersebut bisa disimpan di folder:

app/Libraries/

Yang harus diperhatikan dalam membuat library adalah, setiap nama class harus diawali dengan namespace "Libraries".

<?php
namespace Libraries;

class ValidationForm
{    
    public function __construct()
    {
        
    }
}
